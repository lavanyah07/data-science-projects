# Data Science Projects

This repository contains my Data Science and Machine Learning projects developed using Python.  
The projects demonstrate practical applications of NLP, deep learning, and image analysis.

## Projects Included

1. Sentiment Analysis using Agentic AI  
   - Performs sentiment classification using an AI agent-based approach.

2. OLD2MOD – English Translation using NLP  
   - Converts old English text into modern English using natural language processing techniques.

3. IngreGuard – Minor Project  
   - A machine learning system designed to analyze ingredients and detect potentially harmful components.

4. FoodWrapper – CNN Classification  
   - Uses Convolutional Neural Networks (CNN) to classify food wrapper images.

5. ForgeScan – Image Based Detection  
   - Image-based analysis system for detecting patterns or anomalies using deep learning.

## Technologies Used
- Python  
- Machine Learning  
- Natural Language Processing (NLP)  
- Deep Learning  
- Convolutional Neural Networks (CNN)
